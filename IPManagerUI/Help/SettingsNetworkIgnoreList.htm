Ignore List

The ignore list consist of 2 UI parts. The first part can be found on the main window as an ignore button.
The second part can be found in the settings page as an ignore list and an restore button.
The goal of the ignore list is to ignore certain network interfaces so that they don't show up in your selection list.

For example like the vmware interfaces do.
This way it makes it easier for the end user to quickly select the correct adapter.

Ignore a network interface

To ignore a network interface the button "Ignore" has to be pressed next to the selected network interface.
When this button is pressed, the network interface is actively ignored and won't show up in the selection list.

Restore a network interface

To restore a network interface, we first have to open the settings page.
Once this page is open we can see the ignored list at the bottom of the window.

In the list we can select the network interface we have to restore, this means to be able to select this interface again.
After selecting the interface the button "Restore" has to be pressed.
When this is done the network interface will show up again in the selection list.
