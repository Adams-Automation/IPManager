Network Actions

There are 3 actions we can perform on an network adapter.

  1. Ignore the network adapter
  2. Set IP to the network adapter
  3. Set the network adapter to DHCP

Ignore the network adapter

This means that the network adapter won't show up in list any more and will be ignored.
Also in the future this adapter won't show up. If by mistake this network adapter is ignored, then you can always restore this adapter back into the list.
This can be done in the settings page.

Set IP to the network adapter

After selecting an IP address from the list the IP settings can be set to the network adapter.
By pressing the button a netsh command will be executed to set the IP settings to the correct adapter.

Set the network adapter to DHCP

When this selection is made the network adapter will be set to dhcp mode.
This means that the computer can receive it's IP settings from a dhcp-server.
