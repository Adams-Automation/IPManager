Changing the database for the application.

There are three actions that can be done to change the database for the application :

  • Create a new database at a given location with a given name.
  • Search for a database on your local file system.
  • Set the database location back to default.

One of the main reasons to change the database is when you work togheter with a team, to share the database with the team. This way everyone can use the database and if a new IP setting is filled in everyone can use this setting once they make for instance an intervertion.

Create a new database

To create a new database press the button "New" behind the database location. Once this button is pressed the folder system will open.
In here you can specify the folder to save the database file in and also give the database a name.

Search for a local database

To search for an existing database you can press the three dots behind the database location "...". Once this button is pressed the folder system will open.
In here you can select the database file.

Set the database location to default

To set the default database again you can press the default button behind the database location "Default". This will automatically set the location back to the default file stored by the application. This is the location where the application is stored.
