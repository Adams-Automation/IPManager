Stored IP settings

The stored IP settings can be found on the main window.

In here you can see a list of the IP settings as well as buttons to edit the stored IP's.
The list itself has 4 different columns :

  • Description
  • IP
  • Subnetmask
  • Default gateway

To edit the the IP's the following buttons are available:
  • Add
  • Edit
  • Delete

Add new IP address

To add a new IP address the button "Add" behind the list can be pressed.

By pressing this button, a window will pop-up where the information can be filled in. When all the data is set then the button "Add" can be pressed and the information will be added tot the list.

Edit IP address

To edit an IP address, first select the address that needs to be changed. Then select the "Edit" button.

A new window will pop-up with the information of the selected item. In this window the necessary changes can be done. After changing the information the button "Edit" can be clicked once again. Now the information is stored in the database. The edited data can be found in the list.

Delete IP address

To delete an IP address, first select the address that needs to be deleted. Then select the "Delete" button.

By clicking this button, a delete window will open and shows the information that will be deleted. When the button "Delete" is clicked the data will be deleted from the database.
