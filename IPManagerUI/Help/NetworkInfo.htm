Network Information

The network information show us how the network adapter is currently configured.

For instance in the example above we can see that the network adaptor with name "WiFi" is selected.
The adapter's mode is set to "Dhcp", we can also see the IP V4 address aswel as the subnet mask and default gateway.

When connecting to a network these informations are vital to make sure you're connected to the network and can see the equipment on this network.
